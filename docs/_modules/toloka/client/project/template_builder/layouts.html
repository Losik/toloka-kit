[docs]class ColumnsLayoutV1(BaseLayoutV1):
    """A component for placing content in columns.

    Use it to customize the display of content: set the column width and adjust the vertical alignment of content.

    Attributes:
        full_height ( Optional[Union[BaseComponent, bool]]): Switches the component to column mode at full height and with individual scrolling. Otherwise, the
    		height is determined by the height of the column that is filled in the most.
        items ( Optional[Union[BaseComponent, List[BaseComponent]]]): Columns to divide the interface into.
        min_width ( Optional[Union[BaseComponent, float]]): The minimum width of the component; if it is narrower, columns are output sequentially, one by one.
        ratio ( Optional[Union[BaseComponent, List[Union[BaseComponent, float]]]]): An array of values that specify the relative width of columns. For example, if you have 3 columns, the
    		value [1,2,1] divides the space into 4 parts and the column in the middle is twice as large as the other
    		columns.
    		If the number of columns exceeds the number of values in the ratio property, the values are repeated. For
    		example, if you have 4 columns and the ratio is set to [1,2], the result is the same as for [1,2,1,2].
    		If the number of columns is less than the number of values in the ratio property, extra values are simply
    		ignored.
        validation ( Optional[BaseComponent]): Validation based on condition.
        vertical_align ( Optional[Union[BaseComponent, VerticalAlign]]): Vertical alignment of column content.
    """

[docs]class SidebarLayoutV1(BaseLayoutV1):
    """An option for placing (layout) items, which lets you arrange on a page:

    * The main content block.
    * An adjacent panel with controls.

    The minWidth property sets the threshold for switching between widescreen and compact modes: when the width of the
    layout.sidebar component itself becomes less than the value set by the minWidth property, compact mode is enabled.

    In widescreen mode, the control panel is located to the right of the main block.

    In compact mode, controls stretch to the entire width and are located under each other.

    To add an extra panel with controls, use the extraControls property.

    Attributes:
        content ( Optional[BaseComponent]): Content placed in the main area.
        controls ( Optional[BaseComponent]): Content of the control panel.
        controls_width ( Optional[Union[BaseComponent, float]]): The width of the control panel in widescreen mode. In compact mode, the panel takes up the
    		entire available width. Default: 200 pixels.
        extra_controls ( Optional[BaseComponent]): An additional panel with controls. Located below the main panel.
        min_width ( Optional[Union[BaseComponent, float]]): The minimum width, in pixels, for widescreen mode. If the component width becomes less than the
    		specified value, the interface switches to compact mode. Default: 400 pixels.
